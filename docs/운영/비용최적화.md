## 적용 완료

(stub)
| 항목 | 효과 추정 |
|------|----------|
| WS 폴링 휴장 가드 (시장 OPEN 만 외부 호출) | 외부 API 호출 ~50% ↓ |
| Cache TTL 동적 (휴장 시 core 60s→3m, news 5m→15m) | 휴장 시 외부 호출 ~3배 ↓ |
| Web nginx:alpine (이전 Node serve) | 메모리 70MB → 10MB |
| 정적 자산 1년 immutable cache + gzip | egress 30~50% ↓ |
| CORS 화이트리스트 좁힘 | 외부 봇 차단 |
| IP rate limit (`/auth` 5/분, `/market` 60/분) | 봇 트래픽 차단 |
