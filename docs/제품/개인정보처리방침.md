## Signal Desk 개인정보처리방침

**시행일**: YYYY-MM-DD (출시일)

Signal Desk(이하 "서비스")는 사용자의 개인정보를 중요하게 여기며, 「개인정보 보호법」 등 관련 법령에 따라 다음과 같이 개인정보를 처리합니다.

### 1. 수집하는 개인정보 항목

| 항목 | 수집 시점 | 목적 | 보관 기간 |
|------|----------|------|----------|
| 이메일 | 회원가입 / 소셜 로그인 | 계정 식별, 로그인 | 회원 탈퇴 시까지 |
| 닉네임 | 회원가입 | 표시 이름 | 회원 탈퇴 시까지 |
| 비밀번호 (해시) | 이메일 가입 시 | 인증 | 회원 탈퇴 시까지 |
| 디바이스 푸시 토큰 (Expo Push Token) | 알림 활성화 시 | 푸시 알림 발송 | 알림 비활성화 / 탈퇴 시까지 |
| 사용자 콘텐츠 (관심종목, 보유 종목, AI 추천 로그) | 앱 사용 중 | 개인화된 정보 제공 | 회원 탈퇴 시까지 |

### 2. 수집하지 않는 정보

- 위치 정보 (GPS)

- 광고 식별자 (IDFA / GAID)
- 사용 시간 / 행동 분석 데이터

### 4. 보유 및 이용 기간

회원 탈퇴 시 모든 개인정보는 즉시 파기합니다. 단, 관련 법령에 의해 보존이 필요한 경우 해당 기간 동안 보관합니다.

### 5. 사용자 권리

사용자는 언제든지 다음 권리를 행사할 수 있습니다:
- 개인정보 열람 (앱 내 프로필)
- 개인정보 수정 (앱 내 설정)
- 회원 탈퇴 (앱 내 설정 또는 이메일 요청)
- 푸시 알림 거부 (앱 내 설정)

### 6. 안전 조치

- 비밀번호: BCrypt 해시 저장 (평문 저장 안 함)
- 통신: HTTPS 암호화
- 토큰: JWT (HMAC-SHA256, 30일 만료)
- 인증 시도 제한: 1분당 5회 (브루트포스 방지)
