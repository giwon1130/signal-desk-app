## 기능 요구사항 (Functional)

### Must (반드시)

(stub)
| ID | 요구사항 | 인수 기준 (Acceptance) |
|----|---------|----------------------|
| F-1 | 사용자는 이메일/비밀번호로 가입·로그인할 수 있다 | 가입 → 자동 로그인 / 로그인 → JWT 발급 |
| F-2 | 사용자는 Google OAuth 로 로그인할 수 있다 | iOS native SDK / 웹 GIS 둘 다 |
| F-3 | 사용자는 관심종목을 등록·해제할 수 있다 | 검색 → ☆ 한 탭 → 즉시 반영 + 일괄 해제 |
| F-4 | 사용자는 보유 종목을 등록·수정·삭제할 수 있다 | 매수가 + 수량 → 손익 자동 계산 |
| F-5 | 사용자는 AI 추천 픽을 볼 수 있다 | 오늘 픽 + 진입가/손절가/목표가 |
| F-6 | 사용자는 시장 데이터(지수/뉴스/실험지표)를 볼 수 있다 | KR + US, 휴장 시 다음 거래일 안내 |
| F-7 | 사용자는 장 시작 알림을 받을 수 있다 | 평일 KR 09:00, US 23:30 KST, 분 전 옵션 |
| F-8 | 차트(D/W/M) 토글 + OHLC 툴팁 | 캔들 탭 → 시고저종 표시 |

### Should (가능하면)

(stub)
- 관심종목 급등락 푸시 (서버 발송)
- 알림 히스토리 모달
- Cmd+K 커맨드 팔레트 (웹)

### Could (여유되면)

(stub)
- 모의투자 cash 매매
- 실험지표 detail 모달
- 친구 공유

### Won't (이번엔 안 함)

(stub) 의도적으로 안 만드는 것.
- 자동 매매 — 가이드만 제공, 실행은 사용자
- 실시간 호가창 — 고빈도 트레이더 타깃 아님
- 광고 — 사용자 작은 단계
- 결제/구독 — MVP 무료

## 비기능 요구사항 (Non-functional)

### 성능

(stub)
- 첫 화면 로드 < 2초 (Wi-Fi)
- API 응답 p95 < 500ms
- WS 가격 갱신 5초 이내

### 보안

(stub)
- 비밀번호: BCrypt 해시 (strength 10)
- JWT: HS256, 30일
- IP rate limit: /auth 5/분, /market 60/분
- CORS: 운영 도메인만
- Google OAuth: audience 화이트리스트

### 가용성

(stub)
- 외부 API (KRX/Naver/FRED) 죽어도 fallback 카드 표시
- DB 끊겨도 시장 요약은 살아있음 (메모리 캐시)
- 휴장 시 외부 API 호출 안 함 (비용 절감)

### 접근성

(stub)
- 다크모드 지원
- 시스템 폰트 크기 따름
- 색상만으로 정보 전달 안 함 (색맹 대비)

### 호환성

(stub)
- iOS 16+
- Android 11+
- 웹: Chrome / Safari / Firefox 최신
- 태블릿: 미지원 (모바일 레이아웃)

### 국제화

(stub) 한국어 only. 영어 추가는 추후.
