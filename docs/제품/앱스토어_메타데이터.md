## 긴 설명 (Description — 4,000자)

```
📈 Signal Desk — 한국·미국 주식 투자 대시보드

장 시작 전 점검부터 단타 픽 확인, 보유 종목 익절·손절 가이드까지.
투자자의 하루를 한 화면에서 따라가는 개인 투자 대시보드입니다.

▶ 이런 분께 추천
- 한국·미국 시장을 모두 본다
- 매일 장 시작 전 30분 점검 루틴이 있다
- 단타 + 스윙 둘 다 한다
- 관심종목과 실제 보유를 분리해서 추적하고 싶다

▶ 핵심 기능

1. 오늘의 브리핑
   - KR/US 장 세션 + 휴장·조기종료 자동 인식
   - 뉴스 sentiment (긍정/중립/부정 + 헤드라인)
   - 장전 개인화 브리핑 + 컨텍스트 칩

2. AI 추천 (오늘의 플레이북 + 성적표)
   - 매일 새 단타 픽 (확신도 + 기대 수익률 + 진입가/손절가/목표가)
   - 놓친 픽 / 따라간 픽 비교 — AI 신뢰도 검증
   - 승률·평균수익률·참여율 통계

3. 관심종목 + 보유 분리
   - 관심종목: 추적만 (검색 → ☆ 한 탭)
   - 보유: 매수가·수량 입력 → 손익 자동 계산
   - 한국/미국 통화 표시 (₩ / $)

4. 시장 데이터
   - 코스피/코스닥/NASDAQ/S&P500 D/W/M 캔들 + 거래량 + MA5/20/60
   - 외국인/기관 수급 (KRX)
   - VIX 공포지수 + 실험 지표 (Pentagon Pizza Index 등)

5. 알림
   - 매일 장 시작 알림 (KR 09:00 / US 23:30 KST, 분 전 옵션)
   - 관심종목 ±5% 급등락 푸시 (서버 발송)

▶ 디자인
- 다크모드 지원
- 한국 컨벤션 (빨강=상승, 파랑=하락)
- 정보 밀도 우선 (Yahoo Finance / Toss증권 웹 참고)

▶ 데이터 소스
- 한국: KRX 정보데이터시스템, Naver Finance
- 미국: FRED, CBOE VIX
- 뉴스: Google News RSS

▶ 안전 / 책임
- 모든 추천은 가이드. 매매 결정과 실행은 사용자 본인.
- 서비스는 정보 제공 목적이며 투자 자문이 아닙니다.

▶ 개발 단계
1인 개발자가 만드는 개인 투자 도구입니다. 피드백 환영.
```

## 다음 단계 (사용자 직접)

1. **App Store Connect 에서 앱 생성**
   - https://appstoreconnect.apple.com → My Apps → +
   - Bundle ID: `com.anonymous.signal-desk-app` (또는 변경된 ID)
   - SKU: 임의의 고유 식별자 (예: `signaldesk-ios`)
   - Primary Language: 한국어

2. **이 문서의 내용 복사 / 붙여넣기**
   - 이름 / 부제 / 설명 / 키워드 / 카테고리

3. **스크린샷 촬영**
   - 디바이스에서 직접 촬영 (Cmd+Shift+4 시뮬레이터 / 디바이스 스크린샷)
   - 또는 Figma 등에서 mockup 생성

4. **Privacy Policy 호스팅**
   - 별도 라우트 추가 후 URL 채우기

5. **빌드 업로드**
   - `eas submit -p ios --profile production` (eas.json 의 submit 사용)
   - 또는 Xcode → Window → Organizer → Distribute App
