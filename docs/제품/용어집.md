## 워크스페이스 (사용자 데이터)

| 용어 | 정의 | 코드 타입 | API |
|------|------|----------|-----|
| **관심종목** | 추적만 함. 매수가/수량 없음. | `WatchItem` | `/api/v1/workspace/watchlist` |
| **(실제) 보유** | 매수가·수량 입력해 손익 계산하는 진짜 보유. | `HoldingPosition` | `/api/v1/workspace/portfolio` |
| **모의투자** | 가상 cash 로 매매 시뮬레이션. | `PaperTrading` | `/api/v1/workspace/paper/*` |
| ❌ 즐겨찾기 | **사용 금지** — 과거 잔재. 모두 "관심종목" 으로 통일. |
| ❌ 이상징후 | **사용 금지** — "관심종목 시그널" 로 통일. |
